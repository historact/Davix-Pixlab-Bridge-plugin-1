# Debugging

- Enable **Debug logging** in Settings to capture file logs; use Logs tab to download or clear.
- Enable **Enable logging** to populate DB logs for recent actions.
- Use **Test Connection** to verify Node reachability; inspect HTTP code/body in notice.
- Diagnostics request log fetches Node-side request excerpts for deeper analysis.
- For AJAX issues, inspect browser console; admin debug inline proof posts to JS log when debug enabled.
